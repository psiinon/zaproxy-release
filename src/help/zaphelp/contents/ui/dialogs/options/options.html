Options dialog

The Options dialog allows you to configure ZAP.
It include the following screens:
    Active Scan
    Active Scan Input Vectors
    Anti CSRF Tokens
    API
    Applications
    Authentication
    Breakpoints
    Certificate
    Check for Updates
    Connection
    Database
    Display
    Dynamic SSL Certificate
    Fuzzer
    Keyboard
    Local Proxy
    Passive Scan
    Search
    Spider

Accessed via

     Top level Tools menu'Options ...' menu item
