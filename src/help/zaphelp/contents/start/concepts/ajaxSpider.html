AJAX Spider

The AJAX Spider plugin integrates in ZAP a crawler of AJAX rich sites called Crawljax. You can use it to identify the pages of the targeted site. You can combine it with the spider for better results.

The spider is configured using the Options AJAX Spider screen.

Accessed via

     AJAX Spider tab
     Sites tab'Attack/Spider site' right click menu item
