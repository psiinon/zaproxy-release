Spider tab

The Spider tab shows you the URIs found by the Spider that you have not yet visited.
The Spider can be started from either the Sites tab or the Analysis menu.

Right click menu

Right clicking on a node will bring up a menu which will allow you to:

Find...

This will bring up the Find dialog.

Encode/Decode/Hash...

This will bring up the Encode/Decode/Hash dialog.
If you have highlighted any text then this will be automatically included in the dialog.
