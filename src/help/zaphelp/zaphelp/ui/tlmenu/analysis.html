The Analysis menu

This menu handles spidering and automated scanning.

Spider...

If you have selected one or more URLs in the Sites tab then this will bring up the Spider dialog which allows you to spider the node and all of its children. This will search all of the nodes for new links that you have not visited yet.

Scan All

This will perform an active scan of all of the URLs in the Sites tab.

Scan

This will perform an active scan of all of the selected URLs in the Sites tab.

Scan Policy...

This displays the Scan Policy dialog.

Note that plugins can add additional menu items.
